Security Sensor Deployment

In recent years, the cyberattacks have become more advanced, faster, and more subtle. The evolution of IT services to the Internet, mobile Internet, and public cloud provides attackers with more attacking targets. The security boundaries have more overlaps. APT, 0-day virus, 0-day vulnerability, malware spoofing, sandbox escaping technology are used to bypass the traditional defense. More available hacking toolkits are becoming big challenges.

By the proper deployment of the sensors, DataPot can detect attacks in advance and analyze the stage of the attacks, thus gaining time for security personnel to handle the security risks.

The threats from the external network

✓ Deploy detection sensors (applications)

✓ Find active attackers on the network

✓ Automated security investigations with solid answers

✓ Keep track of threats from all attack phases

✓ Monitor all traffic - internal and external

✓ Analyze logs from security systems, authentication systems and SaaS applications

The threat from the internal network

✓ Deploy detection probes (applications)

✓ Detecting attacks entering the intranet that bypass the border defense to fix the shortcoming of the static defenses.

✓ Continuously monitor the security states of the key assets and detect any threats in the real time.

✓ Continuously look for abnormal behaviors from the internal users and assets and detect the potential risks to reduce the possible loss.
