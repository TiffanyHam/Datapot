Threat Indicator Scaling System

Cyber threats are everywhere. The first step is to exclude the noises and find the useful signals, and expose the threats hidden in the network. The next critical step is to find the most critical and urgent events from the ones with the similar characteristics in the massive data.

In order to improve the accuracy of security information, DataPot establishes a threat indicator scaling system through data analysis and machine learning. This system includes two dimensions: “severity score” and “reliability score”. They are divided into four levels as “serious”, “high-risk”, “medium-risk” and “low-risk”. These four levels provide the basis for priorities.

✓ Attacker dwell time

✓ Events that could jeopardize critical assets in the network

✓ Overall security risk status

✓ Infection rate

✓ Real-time display of security threats in various dimensions (such as network intrusion, abnormal traffic, botnet, system vulnerabilities, website security, etc.) to grasp the current status of security operations.

✓ Network visibility and internal traffic

✓ Classification of attack phases, including (reconnaissance, lateral movement, external access, C&C communication, data exploitation, botnet behavior).

✓ Classification of attack types. The indicator system contains two types of threats and confidence. The indicator application object contains events, event sequences and hosts.
